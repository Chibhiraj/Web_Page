# Full Stack Developer Portfolio (React)

A responsive portfolio site built with **React**, **Vite**, and **React Router**.

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173).

## Build for production

```bash
npm run build
```

Output is in the `dist/` folder. Serve it with any static host.

## Project structure

- `src/` – React app
  - `components/` – Header, Footer, Hero, About, Skills, Projects
  - `pages/` – Home, Contact
  - `data/` – skills and projects content
  - `hooks/` – useInView for scroll animations
- `index.html` – Vite entry
- `css/` – Original static CSS (kept for reference; app uses `src/index.css`)

## Tech stack

React 18, Vite 5, React Router 6.
